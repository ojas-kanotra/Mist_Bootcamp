HTML, or HyperText Markup Language, is the foundational language of the web, serving as the backbone for creating and structuring content on webpages. It provides a systematic way to organize text, images, multimedia, and interactive elements so that web browsers can interpret and display them consistently across devices. Beyond mere presentation, HTML establishes the semantic structure of a webpage, enabling search engines, accessibility tools, and other technologies to understand the content meaningfully. As the cornerstone of web development, it works in tandem with CSS and JavaScript to deliver visually appealing, dynamic, and user-friendly online experiences, forming the essential framework on which the modern internet is built.
===========================================================================

BASIC HTML STRUCTURE
====================
<!DOCTYPE html>                                 Document type declaration - must be first line
<html>                                          Root container element for entire webpage
<head>                                          Metadata section - not displayed to users
<title>Page Title</title>                       Title shown in browser tab and search results
<body>                                          Main content area visible to users
<body style="background-color: RGB(255,0,0);">  Inline CSS styling using RGB color values

HEADINGS
========
<h1>Main Title</h1>                             Largest heading - main page title (SEO important)
<h2>Section Title</h2>                          Major section heading - second most important
<h3>Subsection</h3>                             Subsection within main sections
<h4>Minor Heading</h4>                          Fourth level - detailed categorization
<h5>Small Heading</h5>                          Fifth level - fine-grained organization
<h6>Smallest</h6>                               Least important - rarely used in practice

TEXT CONTENT
============
<p>This is a paragraph.</p>                     Block-level text container with automatic margins
<br>                                            Self-closing line break - forces new line
<hr>                                            Horizontal rule - creates visual section divider

TEXT FORMATTING
===============
<b>Bold text</b>                                Visual bold formatting without semantic meaning
<strong>Important</strong>                      Strong importance - bold with semantic value
<i>Italic text</i>                              Visual italic formatting only
<em>Emphasized</em>                             Emphasized text - italic with semantic meaning
<u>Underlined</u>                               Underlined text (avoid - confuses with links)
<mark>Highlighted</mark>                        Text highlighting - like yellow marker
<small>Fine print</small>                       Smaller text - legal disclaimers, copyright
<big>Bigger text</big>                          Deprecated HTML element - use CSS font-size
<sub>H2O</sub>                                  Subscript - chemical formulas, footnotes
<sup>E=mc2</sup>                                Superscript - mathematical expressions, ordinals
<del>Deleted text</del>                         Strikethrough - shows removed content
<ins>Inserted text</ins>                        Underlined - shows newly added content
<span style="color: red;">Styled</span>         Inline styling container for custom formatting

COMMENTS
========
<!-- This is a comment -->                      Developer notes - invisible to users but in source

LINKS
=====
<a href="https://example.com">Link</a>          Hyperlink to external webpage or resource
<a href="page.html" target="_blank">Link</a>    Opens destination in new browser tab/window
<a href="page.html" target="_self">Link</a>     Opens in same tab (default browser behavior)
<a href="#section1">Jump to section</a>         Internal anchor link to element with id="section1"
<a href="mailto:user@email.com">Email</a>       Creates clickable email link using default client

IMAGES
======
<img src="photo.jpg">                           Self-closing image element (no end tag needed)
<img src="photo.jpg" alt="Person smiling">      Alt text for accessibility and SEO (required!)
<img src="photo.jpg" width="300">               Fixed width in pixels (height auto-scales)
<img src="photo.jpg" height="200">              Fixed height in pixels (width auto-scales)
<img src="photo.jpg" title="My vacation">       Tooltip text displayed on mouse hover
